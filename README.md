# ai_hub_brain

This repository is a public sandbox for student collaboration with agentic workflows.
A group of students is building a student AI hub at Penn State.

Project scope is still being defined. Current focus is building a reliable way for people and agents to brainstorm, organize ideas, and turn selected ideas into shareable artifacts.
